# PicFilter
## 前言
基于Pillow的图片筛选，因为找不到合适的图片筛选工具，就自己写了XD

* 图片去重
* 宽高比筛选
* 宽高值筛选
* 分类为横/竖/方形图片

## 如何使用
安装Pillow库  
`pip install Pillow`  
运行GUI.py，选择操作，输入图片路径和图片输出目录，通过迭代会取出路径内的所有图片

## 注
* 图片去重暂不支持不同大小的相同图片的判断
* 通过筛选的图片会复制到符合的图片目录中
* 图片筛选后的源文件不会删除（图片去重除外）
